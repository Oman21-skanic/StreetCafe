# StreetCafe Website Spec + AI Prompt

This document is a complete specification and ready-to-use AI prompt to generate a simple HTML/CSS/JS website for a fictional cafe brand named StreetCafe. The goal is to present a professional, affordable personal branding site for Indonesian UMKM clients.

## Project Summary

- Brand: StreetCafe
- Business Type: Cafe (coffee + cozy hangout)
- Positioning: Affordable, friendly, modern, and professional
- Target Audience: Indonesian customers, casual hangout seekers, coffee lovers
- Goal: A simple, elegant website that looks premium and trustworthy
- Tech Stack: Plain HTML, CSS, JS (no frameworks)

## Visual Direction

Use a warm, earthy, and modern cafe aesthetic. The design should feel inviting and professional, with strong visual hierarchy and clean spacing.

### Color Palette

- Primary: Coffee Brown #6F4E37
- Secondary: Cream #F5EFE6
- Accent: Caramel #C79054
- Dark Text: Charcoal #2C2C2C
- Light Text: Off-White #FAF7F2
- Highlight: Leaf Green #7C9A6E (used sparingly)

### Typography

- Headings: Playfair Display (serif, elegant)
- Body: Poppins (sans-serif, clean)
- Fallbacks: Georgia, serif / Arial, sans-serif

### Style Notes

- Rounded corners: 12-16px
- Buttons: solid primary, hover darken; secondary outline
- Shadows: soft, subtle
- Spacing: generous padding, clear sections
- Background: cream base, occasional subtle texture or gradient

## Layout Structure (Single Page)

1. Hero Section
   - Large headline: "StreetCafe - Your Cozy Coffee Corner"
   - Subtext: affordable, cozy, and professional
   - CTA buttons: "View Menu" and "Visit Us"
   - Background: warm gradient or cafe image placeholder

2. About Section
   - Short story about StreetCafe
   - Emphasize comfort, affordability, friendly service

3. Signature Menu Section
   - 4-6 featured items with names and prices
   - Use cards with small icons or minimal illustrations

4. Atmosphere Gallery
   - 3-4 image placeholders (use simple blocks or gradients)

5. Testimonials
   - 3 customer quotes (simple cards)

6. Location and Hours
   - Address (fictional)
   - Opening hours
   - Simple map placeholder block

7. CTA Footer
   - "Book a Table" or "Contact Us"
   - Social media links (Instagram, WhatsApp)

## Content (English UI, Indonesian-friendly tone)

Use English for headings and main text, but keep the tone casual and accessible to Indonesian clients. Example: "Affordable coffee, premium vibes". Do not use slang.

### Sample Copy

- Hero headline: "StreetCafe - Affordable Coffee, Premium Vibes"
- Hero subtext: "A cozy hangout with handcrafted coffee and friendly prices."
- About: "StreetCafe is a neighborhood cafe designed for comfort, good conversations, and honest coffee."
- CTA: "View Menu", "Visit Us"

### Sample Menu Items

- Kopi Susu Gula Aren - IDR 18K
- Latte Vanilla - IDR 25K
- Espresso Single - IDR 15K
- Cold Brew Citrus - IDR 28K
- Matcha Cream - IDR 26K
- Chocolate Croissant - IDR 22K

## Interaction and Behavior

- Smooth scroll for nav links
- Hover effects on cards and buttons
- Simple fade-in on scroll (optional, no heavy JS)
- Responsive layout for mobile and desktop

## File Structure

- index.html
- styles.css
- script.js
- assets/ (optional for images)
